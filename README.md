# Hexapod Robot Leg Mechanism Design

This project focuses on the mechanical design of a robotic hexapod leg inspired by biological walking mechanisms.

The objective was to design a stable and functional leg mechanism suitable for multi-servo robotic locomotion.

## Tools Used
- Fusion 360
- Mechanical CAD Modeling

## Project Overview
The hexapod leg was designed with multiple articulated joints to allow controlled motion similar to insect locomotion.

The design focuses on:
- Structural rigidity
- Efficient joint articulation
- Lightweight mechanical structure

## Design Features
- Multi-joint articulated leg mechanism
- Servo motor mounting structure
- Optimized motion range
- Modular design for hexapod robot assembly

## Applications
- Walking robots
- Research on robotic locomotion
- Multi-legged robot prototypes

## Future Work
- Full hexapod robot assembly
- Servo integration for walking gait experiments
- ROS-based locomotion control
